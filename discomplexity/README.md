DISCOMPLEXITY X
===============

"DISCOMPLEXITY"란 "프로그램을 만드는 일이 복잡하지 않다."의 의미를 담은 단어입니다.

[DISCOMPLEXITY X](/README.md) 에는 크게 세가지 패키지로 이루어져 있습니다.

- [CORE](/doc/x/CORE.md): 자료구조 및 입출력과 관련된 라이브러리입니다.
- [GRAPHICS](/doc/x/GRAPHICS.md): 컴퓨터 그래픽스와 관련된 라이브러리입니다.
- [EVENT](/doc/x/EVENT.md): 네트워크와 관련된 라이브러리입니다.
