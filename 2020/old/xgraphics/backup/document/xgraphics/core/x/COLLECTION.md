COLLECTION&lt;T&gt;
===================

컨테이너들의 기반이 되는 추상 클래스로 타입과 유관한 공통의 메서드들을 순수가상함수로 정의하였습니다.

![collection](../../image/core/x/collection.jpg)


| MEMBER             | DESCRIPTION                                              |
| ------------------ | -------------------------------------------------------- |
| each(f: function)  | apply function all container's items                     |
| clear(f: function) | apply function all container's items and clear container |
