CONTAINER
=========

자료구조를 추상화한 가장 기본의 클래스입니다. 타입과 무관하게 공통적으로 필요한 메서드들을 순수 가상 함수로 정의하였습니다.

![container](../../image/core/x/container.jpg)

| MEMBER           | DESCRIPTION                       |
| ---------------- | --------------------------------- |
| size(): uint64   | retrieve the total of items       |
| empty(): boolean | retrieve container's empty status |
| clear()          | clear container                   |
